# 桃桃的单词学习小工具

一款面向小学生的微信小程序，用于英语单词学习、练习和记忆。

## 功能介绍

### 学习模块
- **快速学习** - 随机抽取单词进行学习
- **复习单词** - 复习最近学过的单词
- **单词卡片** - 翻转查看释义和例句，支持音频播放
- **学习进度** - 追踪每日学习数量和掌握情况

### 练习模块
- **汉英练习** - 看中文拼写英文（填空形式）
- **英汉练习** - 看英文选中文释义（选择题）
- **我的单词本** - 从指定单词本练习

### 单词本模块
- 创建多个单词本
- 手动添加单词
- 从 TXT 导入单词（支持格式：单词,音标,释义,例句）
- 删除和编辑单词

## 项目结构

```
word-learning-miniapp/
├── app.js                 # 小程序入口
├── app.json               # 全局配置
├── app.wxss               # 全局样式
├── pages/                 # 页面目录
│   ├── index/             # 首页
│   ├── wordbook/          # 单词本列表
│   ├── wordbook-detail/   # 单词本详情
│   ├── study-select/      # 学习选择
│   ├── study/              # 学习卡片
│   ├── practice/           # 练习选择
│   ├── spelling/           # 汉英练习（拼写）
│   ├── choice/            # 英汉练习（选择）
│   └── profile/           # 个人中心
├── services/              # 服务层
│   ├── audio.js           # 音频服务
│   ├── word.js            # 单词服务
│   └── wordBook.js        # 单词本服务
├── models/                # 数据模型
├── utils/                 # 工具函数
└── custom-tab-bar/        # 自定义导航栏
```

## 技术栈

- 微信小程序原生开发

## 快速开始

1. 克隆项目
```bash
git clone https://github.com/luokpX/word-learning-miniapp.git
```

2. 在微信开发者工具中导入项目

3. 使用真实小程序 AppID 或测试号进行开发

## 发布

1. 在微信开发者工具中点击"上传"
2. 填写版本号和备注
3. 在微信公众平台提交审核
4. 审核通过后发布
